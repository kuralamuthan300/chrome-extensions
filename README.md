## 🛠️ General Installation Guide

To run any of these extensions locally in your Chrome browser:

1.  **Clone the Repository**:
    ```bash
    git clone [your-repo-link]
    cd chrome-extension
    ```

2.  **Open Chrome Extensions Page**:
    Navigate to `chrome://extensions/` in your browser.

3.  **Enable Developer Mode**:
    Toggle the switch in the top-right corner to **Developer mode**.

4.  **Load Unpacked Extension**:
    - Click the **Load unpacked** button.
    - Navigate to the specific extension folder (e.g., `to-do` or `eye-well-being`) and click **Select**.

5.  **Pin for Easy Access**:
    Click the puzzle icon (Extensions) in your browser toolbar and pin the extension.
